# Data Analytics Portfolio

This repository is a structured, scalable portfolio for my data analytics work.
It is designed to host projects across Python, SQL, Power BI, and general
analytics workflows. The structure supports reproducibility, documentation, and
version control for end-to-end analytics projects.

## 📁 Repository Structure

- **projects/** – Individual analytics projects (Python, SQL, BI, ML, etc.)
- **shared/** – Reusable datasets, scripts, themes, and utilities
- **notebooks/** – Jupyter notebooks for exploration and analysis
- **documentation/** – Screenshots, diagrams, templates, and notes
- **LICENSE** – Open-source license for this portfolio

## 🎯 Purpose

This portfolio will showcase:
- data cleaning and transformation
- exploratory data analysis (EDA)
- statistical analysis
- data visualisation
- BI dashboards
- SQL workflows
- Python analytics
- reproducible project structures

Projects will be added over time.
