# Projects

This folder will contain individual data analytics projects. Each project will
be self-contained and follow a consistent structure:

```mermaid
Project Name
    1. Data
        Raw         # Original datasets (if they arnt to big to share)
        Cleaned     # Processed dataserts ready for analysis (if they arnt to big to share)
    2. Notebooks    # Jupyter notebooks for EDA, modelling or reporting
    3. Scripts
        Python      # Python ETL, cleaning feature engineering
        SQL         # SQL queries, transformations, views
    4. Outputs      # Power BI, Tableau, or static visualisations
    README.md
```
Projects will be added here as they are developed.
